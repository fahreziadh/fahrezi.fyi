### 1. Program AWS Active

Awalnya saya tidak tahu tentang program ini secara langsung. Sebaliknya, saya menemukannya melalui [halaman AWS Active Credits](https://aws.amazon.com/startups/credits) 
![aws-startups](/img/aws-active.webp "")

Ada dua program: 
- AWS Active Founders ($1k)
- AWS Active Portfolio (hingga $100k). 

Tentu saja, mendapatkan kredit Active Founder ($1k) jauh lebih mudah dan cepat, karena kita hanya perlu membuat akun AWS dan memasukkan informasi tentang produk yang ingin kita buat.
Namun, kali ini saya akan membahas cara mendapatkan hingga $100k 👑 melalui program AWS Active Portfolio. 

Untuk mendapatkannya, kita perlu berada dalam daftar penyedia aktif AWS berikut: [Daftar Active Provider](https://aws.amazon.com/activate/portfolio-detail/)
![aws-portfolio](/img/aws-active-featured.webp "")

Ya, itulah aturannya. Cara masuknya adalah sebagai individu atau sebagai tim/perusahaan dengan mendaftar atau bahkan diundang langsung. Kemudian kita akan otomatis bisa mengajukan kredit AWS hingga $100k.

Tapi, jangan khawatir, saya sudah melakukan riset tentang hal ini dan menemukan bahwa program yang paling cocok untuk kita **founder solo** atau tim **startup kecil** adalah program Nvidia Inception yang saya sebutkan di awal. Jadi, bagaimana saya berhasil mendaftar? Mari kita bahas lebih lanjut :D.

`Saya ingin memberikan sedikit disclaimer sebelum melanjutkan: mohon hanya mendaftar untuk bergabung jika Anda memiliki proyek yang serius. Meskipun belum ada profit/pendapatan, kita perlu mempersiapkan dengan matang beberapa hal terkait produk yang kita buat, seperti deskripsi singkat proyek, bidangnya, dan target pasar. Jika kita tidak serius, mungkin lebih baik hanya mendaftar program AWS Active Founder, sebelum aplikasi kita ditolak.`

### 2. Aplikasi Program Nvidia Inception

#### 2.1 Mengisi formulir aplikasi

Semuanya dimulai pada 13 Mei 2024, ketika saya mendaftar untuk program ini. Saya tidak berharap banyak karena saya pernah mendaftar untuk program serupa beberapa kali sebelumnya dan kebanyakan ditolak.
![nvidia-intro](/img/13th-may-email.png "")

Untuk proses aplikasi, sebenarnya mirip dengan mendaftar AWS Active Founder ($1k). Kita diminta mengisi data pribadi, profil proyek/perusahaan seperti masalah & solusi yang akan kita selesaikan dengan produk kita, kompetitor kita, beberapa informasi pribadi terkait pengalaman kerja (saya tidak tahu kenapa), dan juga teknologi Nvidia yang akan digunakan dalam proyek.

#### 2.2 Menerima respon + follow up informasi detail

Singkat cerita, saya menerima email dari tim Nvidia Inception 14 hari kemudian dan diminta menjawab beberapa pertanyaan. Saya akan menyertakan gambar dengan jawaban saya yang sudah diisi, yang bisa Anda gunakan sebagai referensi :D
![email-response](/img/email-response.png "")

Dan akhirnya, setelah 1 bulan, saya menerima email ini 🎉🎉🎉

![nvidia-response](/img/nvidia-response.png "")

### 3. Mengajukan AWS Active Portfolio

Setelah menerima respon dari Nvidia, saya mulai mengajukan program AWS Active Portfolio. Tapi, sebelum saya melakukannya, kita perlu meminta surat manfaat ke Nvidia Inception melalui website mereka. Saya melakukannya dan menerima respon dalam beberapa hari.

![aws-benefits](/img/nvidia-inception-benefits.png "")

Setelah itu, saya langsung mengajukannya ke konsol AWS yang sudah terhubung dengan akun AWS aktif. Saya membuat pengajuan dan memasukkan org id yang telah diberikan melalui email. Tapi sebelum itu, perhatikan hal-hal berikut (sangat penting):

- Di field valuasi, jangan pilih "No Valuation" (saya mencobanya karena saya pikir produk saya belum menguntungkan, tapi aplikasi ditolak, mungkin karena dianggap tidak serius. Untungnya saya bisa mengajukan kembali).
- Pastikan email yang digunakan untuk aplikasi NVIDIA Inception sama dengan email yang digunakan untuk akun aktif AWS dan konsol AWS.
- Jangan menyebarkan rahasia seperti org-id, menjual akun, atau melakukan hal-hal tidak bertanggung jawab lainnya.

Anda mungkin bertanya, mengapa kita tidak boleh memilih "no valuation"? Yang saya tahu adalah Valuasi != Profit. Meskipun produk kita belum ada profit, aset kita seperti produk kita seharusnya tetap memiliki nilai, bukan?
Untuk sisanya, kita tinggal menunggu, dan jika aplikasi diterima, kita akan menerima email seperti ini.

### 4. Menukarkan Kredit AWS

Kemudian selamat: Anda akan mendapatkan $10k gratis di konsol AWS Anda.

![nvidia-response](/img/aws-response-credits.png "")

Kredit Activate diterapkan secara bertahap hingga maksimum kredit Activate sebesar $100.000 (catatan: kredit tidak menumpuk). Kredit Anda harus meningkat seiring waktu sampai Anda mencapai maksimum seumur hidup $100.000. Oleh karena itu, Anda tidak dapat mengajukan paket yang lebih kecil dari atau sama dengan paket yang sebelumnya diaktifkan. 

Tidak hanya itu, Anda juga bisa mendapatkan [AWS Exclusive Offer](https://aws.amazon.com/startups/learn/aeo) seperti kredit gratis dari Vercel, Datadog, dan banyak item gratis dan diskon lainnya karena Anda sudah menjadi anggota aktif AWS.
